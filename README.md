# 🎬 TV Show Takedown: Who’s Running the Streaming Wars?

A data-driven project for **StreamVerse** to find out which TV shows are worth investing in 📊

## 🚀 Overview
Streaming platforms are full of shows, but not all of them deserve attention, budget, or promotion.  
This project analyzes real TV show data to understand what makes a show successful and how StreamVerse can make smarter content decisions.

## 📂 Dataset
The dataset includes features like:

- show_name
- genres
- release_year
- number_of_seasons
- rating
- votes
- popularity
- language
- country_of_origin
- status

## 🎯 Objectives

### 1. 🎭 The Genre Puzzle
Analyze how genres perform in terms of ratings, popularity, and audience reception.

### 2. 📺 The Longevity Paradox
Check whether shows with more seasons are actually loved more by audiences.

### 3. 💎 The Paradox of Visibility
Find hidden gems, blockbusters, overhyped shows, and forgotten content.

### 4. ⏳ The Nostalgia Question
Study how TV show quality has changed over time across different eras and genres.

### 5. 🤖 The Success Predictor
Build a machine learning model to predict show ratings and identify key success factors.

### 6. 🍿 The Taste Maker
Create a recommendation engine that suggests similar shows viewers may enjoy.

## 🛠️ Tools Used
- Python 🐍
- Pandas
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## 📈 Outcome
This project helps StreamVerse:
- invest in stronger genres
- spot underrated shows
- understand audience behavior
- predict success more accurately
- improve recommendations

## ✨ Goal
Use **data, not guesses**, to decide who’s really winning the streaming wars.
